
# Sugarfit Java Service Starter

Production-ready Spring Boot microservice template.

## Features
- Layered architecture
- Validation & global exception handling
- Request tracing with MDC
- OpenAPI (Swagger)
- Actuator monitoring
- Docker & Kubernetes ready
- GitHub Actions CI
- Integration test

## Run
mvn clean install
mvn spring-boot:run

Swagger:
http://localhost:8080/swagger-ui.html
